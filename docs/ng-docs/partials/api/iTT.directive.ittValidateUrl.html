ittValidateUrl
directive in module iTT

Description

Directive used on url inputs to allow custom validation

Dependencies

  • $q
  • ngModel
  • errorSvc
  • youtubeUrlService
  • ittUtils
  • dataSvc

Usage

as element:
<itt-validate-url
       item="{Object}">
</itt-validate-url>
as attribute
<ANY itt-validate-url
     item="{Object}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
itemObject

The item that the url we are validating belongs to

Example

<input type="url" itt-validate-url item="item"/>
