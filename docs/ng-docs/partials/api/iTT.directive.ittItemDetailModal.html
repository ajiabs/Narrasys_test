ittItemDetailModal
directive in module iTT

Description

For opening modals with event objects. Currently resides in player.html and is passed the item data via appState. Modal is invoked via forceModal() $scope method in ittItem. forceModal() accepts a bool as param, and sets the item on the appState. The modal transition is animated based upon the boolean param passed to forceModal()

Dependencies

  • appState

Usage

as attribute
<ANY itt-item-detail-modal
     item="{Object}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
itemObject

Event object to display in modal
