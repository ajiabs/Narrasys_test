ittIframe
directive in module iTT

Description

Directive used to display iframed content

Dependencies

  • appState

Usage

as element:
<itt-iframe
       src="{String}"
       [contenttype="{String}"]
       item="{Object}">
</itt-iframe>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
srcString

The Source URL for the iframe

contenttype
(optional)
String

The mime type of the iframed content

itemObject

The Item object, aka an Event from the Database

Example

<itt-iframe src="<url>" contenttype="text/html" item="<itemObject>"></itt-iframe>
