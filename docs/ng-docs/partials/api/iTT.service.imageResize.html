imageResize
service in module iTT

Description

Resize images and convert File Objects into images. Used in practice to resize images client side prior to uploading to AWS. Basic Idea is to resize images client side with HTML5 Canvas. The resized image is converted back into a File object so it can be passed to the AWS service. source
