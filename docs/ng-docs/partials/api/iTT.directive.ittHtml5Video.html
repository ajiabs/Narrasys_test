ittHtml5Video
directive in module iTT

Description

Directive used to render html5 videos

Dependencies

  • $timeout
  • html5PlayerManager
  • appState

Usage

as element:
<itt-html5-video
       video-url="{String | Object}"
       main-player="{Boolean}"
       player-id="{String}">
</itt-html5-video>
as attribute
<ANY itt-html5-video
     video-url="{String | Object}"
     main-player="{Boolean}"
     player-id="{String}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
videoUrlString Object

The URL to the youtube video. can be a string (for embeds), or an object (for the main video).

mainPlayerBoolean

Whether or not the player is to be used as the main video or embed.

playerIdString

ID either main video asset ID or event ID

Example

//for the main player
<itt-html5-video video-url="<OBJECT or URL>" main-player=true player-id="<ID>"></itt-html5-video>
