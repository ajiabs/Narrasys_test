ittNarrativeEditor
directive in module iTT

Description

A presentational component used to edit narratives

Usage

as element:
<itt-narrative-editor
       narrative="{Object}"
       customers="{Array}"
       container-info="{customerId:string, containerId:string, name:string}"
       on-done="{Function}"
       on-update="{function}">
</itt-narrative-editor>
as attribute
<ANY itt-narrative-editor
     narrative="{Object}"
     customers="{Array}"
     container-info="{customerId:string, containerId:string, name:string}"
     on-done="{Function}"
     on-update="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
narrativeObject

the narrative to edit

customersArray

an array of customers associated with the narrative

containerInfocustomerId:string, containerId:string, name:string

container specific info to set on the narrative

onDoneFunction

output to call when on cancel.

onUpdatefunction

output to call when saving.

Example

//containers page
<itt-narrative-editor
       customers="customers"
       container-info="{containerId: container._id, customerId: container.customer_id, name: container.name}"
       on-done="toggleNarrativeModal()"
       on-update="postNewNarrative(data)">
     </itt-narrative-editor>

     //narrative show/index pages
     <itt-narrative-editor
       customers="$ctrl.selectedCustomer"
       narrative="$ctrl.narrativeToEdit"
       on-done="$ctrl.closeAddOrEditModal()"
       on-update="$ctrl.addOrUpdateNarrative(n)">
     </itt-narrative-editor>
