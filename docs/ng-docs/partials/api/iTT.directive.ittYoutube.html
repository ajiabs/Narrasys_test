ittYoutube
directive in module iTT

Description

Directive used to render the actual youtube iframe and link YT instances with the youTubePlayerManager service source

Dependencies

  • $timeout
  • $scope
  • youTubePlayerManager
  • youtubeSvc

Usage

as element:
<itt-youtube
       video-url="{String}"
       main-player="{Boolean}"
       player-id="{String}">
</itt-youtube>
as attribute
<ANY itt-youtube
     video-url="{String}"
     main-player="{Boolean}"
     player-id="{String}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
videoUrlString

The URL to the youtube video

mainPlayerBoolean

Whether or not the player is to be used as the main video or embed.

playerIdString

ID either main video asset ID or event ID, used to set PID of YT Instance inside _players Object

Example

//for the main player
<itt-youtube video-url="path/to/url" main-player="true" player-id="<ID>"></itt-youtube>
